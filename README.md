# TALENCA LEAGUE

## Overview
A Command line application which process a text file with game results and returns a ranking of the teams.

## Prerequisites
- .NET Core SDK (.NET 7)

## Usage
Just go to .\Main and run

```bash
cd .\Main\
dotnet run
```

## Test
Just go to .\Main and run

```bash
cd .\Main.Test\
dotnet test
```
